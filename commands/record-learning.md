# /record-learning

Fallback phrase: `Record learning: subagents need explicit feature context before implementation`

Required skill: `superpowers:project-memory`

Use `project-memory` in record mode. Classify the lesson and update workflow memory files only: `FEATURE_CONTEXT.md`, `BUG_PATTERNS.md`, `SPEC_INDEX.md`, or `ACTIVE_CONTEXT.md`. If the lesson affects foundational architecture, structure, conventions, testing, integrations, or concerns, record a proposed update or delegate source inspection to `building-codebase-memory`.
